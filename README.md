# CHIP8 Emulator

Written by Aiden Pyle

## Instructions

| Bytes | Operation |
| ---- | ----------- |
| 00E0 | Clears the screen |
| 1NNN | Jumps to address NNN |
| 6XNN | Set register X to value NN |
| 7XNN | Add value NN to register X |
| ANNN | Set index register to NNN |
| DXYN | Draw N bytes at (X, Y) |
